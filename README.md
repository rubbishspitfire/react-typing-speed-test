# React Typing Speed Test

A typing speed test app built with React and JavaScript.

## Features
- Live WPM calculation
- Accuracy tracking
- Mistake counter
- Difficulty levels
- Character-by-character highlighting
- Progress bar
- Dark mode
- Local leaderboard with localStorage

## Tech Stack
- React
- JavaScript
- CSS
- Vite

## Run Locally
```bash
npm install
npm run dev
