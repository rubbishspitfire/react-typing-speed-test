# React Typing Speed Test

A polished typing speed test built with React, JavaScript, CSS, and Vite.

## Live Demo
https://rubbishspitfire.github.io/react-typing-speed-test/

## Features
- Real-time WPM tracking
- Accuracy and mistake tracking
- Character-by-character highlighting
- Difficulty levels
- Progress bar
- Dark mode
- Sound toggle
- Persistent local leaderboard
- Responsive layout

## Tech Stack
- React
- JavaScript
- CSS
- Vite
- localStorage

## Getting Started

```bash
npm install
npm run dev
